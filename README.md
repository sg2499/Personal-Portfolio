# Shailesh Gupta Portfolio

Deployable Vite + React + Tailwind portfolio website.

## Local run

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to a GitHub repo.
2. Open Vercel.
3. Import the repo.
4. Framework preset: Vite.
5. Deploy.
